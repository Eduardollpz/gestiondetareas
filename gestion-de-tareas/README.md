# Gestion de Tareas

## Descripción
Gestion de Tareas es una aplicación simple para gestionar tareas diarias. Permite a los usuarios agregar tareas, marcarlas como completadas y eliminarlas de la lista.

## Estructura del Proyecto
El proyecto tiene la siguiente estructura de archivos:

```
gestion-de-tareas
├── index.html          # Documento HTML principal
├── src
│   └── app.js         # Código JavaScript para gestionar tareas
├── styles
│   └── styles.css     # Estilos CSS para la aplicación
├── .gitignore          # Archivos y directorios a ignorar por Git
├── package.json        # Configuración de npm
└── README.md           # Documentación del proyecto
```

## Instalación
1. Clona el repositorio:
   ```
   git clone <URL_DEL_REPOSITORIO>
   ```
2. Navega al directorio del proyecto:
   ```
   cd gestion-de-tareas
   ```
3. Instala las dependencias (si las hay):
   ```
   npm install
   ```

## Uso
1. Abre el archivo `index.html` en un navegador web.
2. Agrega tareas en el campo de entrada y haz clic en el botón para añadirlas a la lista.
3. Marca las tareas como completadas o elimínalas según sea necesario.
